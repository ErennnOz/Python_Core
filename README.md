# 🐍 Python Codes – Practice Repository

## 📌 Purpose

This repository aims to improve fundamental programming logic and algorithmic thinking by solving Python problems at various levels.

The problems included in this repository are taken from a resource containing **140+ Python programs ranging from basic to advanced**, and are implemented for practice purposes.

Through consistent problem-solving, this repository serves as a practice space to:

- Reinforce Python syntax
- Improve algorithmic thinking
- Learn fundamental and intermediate programming concepts through hands-on practice

---

## 📂 Content

All implementations are located in the `main.py` file and are organized as numbered problems (cases).

Main topics covered in this repository include:

- Basic Python programs (output, input, arithmetic operations)
- Conditional statements (if-else)
- Loops (for, while)
- Number-based problems (prime numbers, Armstrong numbers, Fibonacci, factorial)
- List and array operations
- String manipulations
- Mathematical computations (LCM, HCF, logarithms, etc.)
- Recursive solutions
- Matrix operations
- Small-scale algorithm problems

Each problem is handled individually to support the learning process.
